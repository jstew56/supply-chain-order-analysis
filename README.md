# Supply Chain Order Analysis (SQL + Excel)

## Overview

This project analyzes supply chain order data using SQL joins across a star schema model and transforms the results into an Excel Pivot Table dashboard for operational reporting.

The goal was to consolidate customer, order, product, and supplier data into a unified view to support revenue and performance analysis.

---

## Key Skills Demonstrated

- Multi-table INNER JOIN  
- Primary & foreign key relationships  
- Fact vs dimension modeling  
- Star schema understanding  
- Excel Pivot Tables  
- Data aggregation and summarization  
- Business-focused reporting  

---

## Tools Used

- SQL (Snowflake)
- Microsoft Excel

---

## Project Outcome

This project demonstrates the ability to work with relational datasets, structure SQL queries across multiple tables, and translate operational data into clear, decision-ready insights.
